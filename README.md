# Turing Visualizer

An interactive visualizer for Turing machines.

You can check it out [here](https://grabczak.github.io/turing-visualizer/)

## Quick Start

### Prerequisites

[Node.js](https://nodejs.org) must be installed on the system.

### Build and Run

Clone the repository and run:

```bash
npm install
npm run dev
```
